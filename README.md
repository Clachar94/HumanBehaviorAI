## Descripción del proyecto
Sistema de detección de manipulación digital y comportamiento artificial en plataformas 
digitales, desarrollado mediante técnicas de Big Data e Inteligencia Artificial. 
El sistema analiza patrones de comportamiento de usuarios para identificar cuentas 
automatizadas (bots) mediante algoritmos de detección de anomalías y clasificación.
